# Holberton School - Low Level Programming & Algorithms

Ce repository regroupe les projets de programmation bas niveau en C : bases du langage, gestion mémoire, pointeurs, structures de données, débogage et analyse dynamique.

## Sous-dossiers

| Dossier | Description |
|---|---|
| `hello_world/` | Premiers programmes C, compilation |
| `variables_if_else_while/` | Variables, conditions, boucles |
| `functions_nested_loops/` | Fonctions et boucles imbriquées |
| `more_functions_nested_loops/` | Fonctions et boucles imbriquées (approfondissement) |
| `pointers_arrays_strings/` | Pointeurs, tableaux, chaînes de caractères |
| `argc_argv/` | Arguments de la ligne de commande |
| `malloc_free/` | Allocation dynamique de mémoire |
| `more_malloc_free/` | Allocation dynamique de mémoire (approfondissement) |
| `structures_typedef/` | Structures et `typedef` |
| `function_pointers/` | Pointeurs de fonctions |
| `recursion/` | Fonctions récursives |
| `variadic_functions/` | Fonctions à nombre variable d'arguments |
| `singly_linked_lists/` | Listes chaînées simples |
| `doubly_linked_lists/` | Listes chaînées doubles |
| `hash_tables/` | Tables de hachage |
| `file_io/` | Entrées/sorties fichiers |
| `intro_debugging/` | Introduction au débogage avec `gdb` |
| `dynamic_analysis/` | Analyse dynamique avec `valgrind` |
| `benchmarking/` | Mesures de performance |
| `secure_data_handling/` | Manipulation sécurisée des données |
| `ai_memory_visualizer_valgrind_tracer/` | Bundle de programmes de visualisation mémoire |
| `simple-calculator/` | Petite calculatrice en C |
