# 0x04. C - More functions, more nested loops

## Description
Approfondissement des fonctions et boucles imbriquées : tests de caractères, tables ASCII, FizzBuzz, triangles, facteurs premiers.

## Fichiers

| Fichier | Description |
|---|---|
| `0-isupper.c` | Vérifie si un caractère est en majuscule |
| `1-isdigit.c` | Vérifie si un caractère est un chiffre |
| `2-mul.c` | Multiplie deux entiers |
| `3-print_numbers.c` | Affiche les nombres de 0 à 9 |
| `4-print_most_numbers.c` | Affiche les nombres de 0 à 9 sauf 2 et 4 |
| `5-more_numbers.c` | Affiche les nombres de 0 à 14 |
| `6-print_alphabet.c` | Affiche l'alphabet en minuscule |
| `7-print_tebahpla.c` | Affiche l'alphabet en minuscule à l'envers |
| `8-print_last_digit.c` | Affiche le dernier chiffre d'un nombre |
| `100-*.c` | Exercices avancés : FizzBuzz, triangles |
| `101-print_number.c` | Affiche un nombre avec `_putchar` |
