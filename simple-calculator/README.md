#README

##Compile command
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 calculator.c -o calculator

##Run instructions
To run the program, simply use ./calculator in your prefered terminal

##Supported operations
With this amazing program you can add, substract, divide and multiply

##Numeric behavior
Whenever you want to divide, we'll cast an integer into a float with 1 decimal after comma.

##known limitations 
If the user enters other inputs than 0 to 4, the program will stop, it doesnt handle the case where user enter letters or other numbers.
