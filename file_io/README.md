# 0x0D. C - File I/O

## Description
Manipulation des entrées/sorties fichiers en C avec les appels système `open`, `read`, `write`, `close`, jusqu'à la réimplémentation de la commande `cp`.

## Fichiers

| Fichier | Description |
|---|---|
| `0-read_textfile.c` | Lit un fichier texte et l'affiche sur la sortie standard |
| `1-create_file.c` | Crée un fichier avec un contenu donné |
| `2-append_text_to_file.c` | Ajoute du texte à la fin d'un fichier |
| `3-cp.c` | Réimplémentation de la commande `cp` |
| `main.h` | Prototypes des fonctions du projet |
