# 0x1A. C - Hash tables

## Description
Implémentation d'une table de hachage en C : création, fonction de hachage `djb2`, gestion des collisions, insertion, récupération, affichage et suppression.

## Fichiers

| Fichier | Description |
|---|---|
| `hash_tables.h` | Déclaration de la structure `hash_table_t` et prototypes |
| `0-hash_table_create.c` | Crée une table de hachage |
| `1-djb2.c` | Implémente la fonction de hachage `djb2` |
| `2-key_index.c` | Calcule l'index d'une clé dans le tableau |
| `3-hash_table_set.c` | Ajoute ou met à jour un élément dans la table |
| `4-hash_table_get.c` | Récupère la valeur associée à une clé |
| `5-hash_table_print.c` | Affiche la table de hachage |
| `6-hash_table_delete.c` | Supprime une table de hachage et libère la mémoire |
