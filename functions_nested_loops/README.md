# 0x03. C - Functions, nested loops

## Description
Création de fonctions et utilisation de boucles imbriquées : alphabet, tests de caractères, table de multiplication, suite de Fibonacci.

## Fichiers

| Fichier | Description |
|---|---|
| `0-putchar.c` | Affiche le caractère `_` |
| `1-alpha.c` | Affiche l'alphabet en minuscule |
| `2-alpha.c` | Affiche l'alphabet, une lettre par ligne |
| `3-isupper.c` | Vérifie si un caractère est en majuscule |
| `4-islower.c` | Vérifie si un caractère est en minuscule |
| `5-isalpha.c` | Vérifie si un caractère est une lettre |
| `6-abs.c` | Calcule la valeur absolue d'un entier |
| `7-mult_table.c` | Affiche la table de multiplication de 0 à 9 |
| `100-*.c` | Variantes avec bases/paramètres différents |
| `104-fibonacci.c` | Calcule les termes de la suite de Fibonacci |
