# 0x06-0x09. C - Pointers, Arrays, Strings

## Description
Manipulation intensive des pointeurs, tableaux et chaînes de caractères en C : réimplémentation de fonctions de la libc (`memset`, `strcat`, `memcpy`, `strncat`, `strchr`, `strlen`, `strncpy`, `strcmp`, `strspn`, `strpbrk`, `strstr`, `atoi`...) et petits algorithmes (`rev_array`, `rot13`, `keygen`).

## Fichiers

| Fichier | Description |
|---|---|
| `0-memset.c` | Réimplémentation de `memset` |
| `1-strcat.c` | Réimplémentation de `strcat` |
| `2-memcpy.c` | Réimplémentation de `memcpy` |
| `3-strncat.c` | Réimplémentation de `strncat` |
| `4-swap_int.c` | Échange deux entiers via pointeurs |
| `5-strchr.c` | Réimplémentation de `strchr` |
| `6-strspn.c` | Réimplémentation de `strspn` |
| `7-strpbrk.c` | Réimplémentation de `strpbrk` |
| `8-strstr.c` | Réimplémentation de `strstr` |
| ... | Autres exercices sur pointeurs, tableaux 2D, chaînes (0 à 104) |
| `104-print_buffer.c` | Affiche le contenu d'un buffer en hexadécimal |
