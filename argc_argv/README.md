# 0x02. C - Arguments, opportunities

## Description
Manipulation des arguments passés à un programme C (`argc`, `argv`) : affichage, calculs, conversion.

## Fichiers

| Fichier | Description |
|---|---|
| `0-whatsmyname.c` | Affiche le nom du programme |
| `1-args.c` | Affiche le nombre d'arguments passés |
| `2-args.c` | Affiche chaque argument sur une ligne |
| `3-mul.c` | Multiplie deux nombres passés en argument |
| `4-add.c` | Additionne des nombres positifs |
| `100-atoi.c` | Implémentation de `atoi` sur les arguments |
