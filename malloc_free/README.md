# 0x0A. C - malloc, free

## Description
Introduction à l'allocation dynamique de mémoire avec `malloc` et `free` : création de tableaux, duplication de chaînes, concaténation, grilles 2D.

## Fichiers

| Fichier | Description |
|---|---|
| `0-create_array.c` | Crée un tableau de `char` initialisé à une valeur |
| `1-strdup.c` | Réimplémentation de `strdup` |
| `2-str_concat.c` | Concatène deux chaînes de caractères |
| `3-alloc_grid.c` | Alloue dynamiquement une grille 2D |
| `4-free_grid.c` | Libère la mémoire d'une grille 2D |
