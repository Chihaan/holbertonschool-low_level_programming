# 0x1D. C - Intro to debugging

## Description
Exercices de débogage de programmes C fournis avec des bugs, à corriger à l'aide de `gdb`.

## Fichiers

| Fichier | Description |
|---|---|
| `multiplication_table.c` | Affiche une table de multiplication (contient un bug à corriger) |
| `row_sums.c` | Calcule la somme des lignes d'une matrice (contient un bug à corriger) |
| `sum_to_n.c` | Calcule la somme de 1 à n (contient un bug à corriger) |
