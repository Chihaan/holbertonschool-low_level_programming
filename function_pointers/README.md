# 0x0E. C - Function pointers

## Description
Utilisation des pointeurs de fonctions : affichage de noms, itération sur tableau, opérations arithmétiques dynamiques via une petite calculatrice.

## Fichiers

| Fichier | Description |
|---|---|
| `0-print_name.c` | Appelle une fonction via un pointeur de fonction |
| `1-array_iterator.c` | Exécute une fonction sur chaque élément d'un tableau |
| `2-int_index.c` | Cherche l'index d'un élément via une fonction de comparaison |
| `3-calc.h` / `3-op_functions.c` / `3-get_op_func.c` | Petite calculatrice basée sur des pointeurs de fonctions |
| `function_pointers.h` | Prototypes des fonctions du projet |
