# 0x12. C - Doubly linked lists

## Description
Manipulation de listes chaînées doubles en C : affichage, longueur, ajout, suppression, recherche et somme des éléments.

## Fichiers

| Fichier | Description |
|---|---|
| `lists.h` | Déclaration de la structure `dlistint_t` et prototypes |
| `0-print_dlistint.c` | Affiche tous les éléments d'une liste `dlistint_t` |
| `1-dlistint_len.c` | Calcule le nombre d'éléments d'une liste |
| `2-add_dnodeint.c` | Ajoute un nouveau nœud au début de la liste |
| `3-add_dnodeint_end.c` | Ajoute un nouveau nœud à la fin de la liste |
| `4-free_dlistint.c` | Libère une liste `dlistint_t` |
| `5-get_dnodeint.c` | Retourne le nœud à un index donné |
| `6-sum_dlistint.c` | Calcule la somme des éléments de la liste |
| `7-insert_dnodeint.c` | Insère un nouveau nœud à un index donné |
| `8-delete_dnodeint.c` | Supprime le nœud à un index donné |
