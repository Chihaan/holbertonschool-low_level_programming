# 0x0B. C - more malloc, free

## Description
Approfondissement de l'allocation dynamique de mémoire : vérification des erreurs `malloc`, concaténation de chaînes, `calloc`, tableaux dynamiques.

## Fichiers

| Fichier | Description |
|---|---|
| `0-malloc_checked.c` | `malloc` avec vérification et sortie en cas d'échec |
| `1-string_nconcat.c` | Concatène deux chaînes en limitant le nombre de caractères de la seconde |
| `2-calloc.c` | Réimplémentation de `calloc` |
| `3-array_range.c` | Crée un tableau contenant une plage de nombres |
