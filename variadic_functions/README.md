# 0x10. C - Variadic functions

## Description
Utilisation de `stdarg.h` pour créer des fonctions acceptant un nombre variable d'arguments.

## Fichiers

| Fichier | Description |
|---|---|
| `0-sum_them_all.c` | Additionne un nombre variable d'entiers |
| `1-print_numbers.c` | Affiche un nombre variable d'entiers séparés par un séparateur |
| `2-print_strings.c` | Affiche un nombre variable de chaînes séparées par un séparateur |
| `3-print_all.c` | Affiche différents types d'arguments selon un format |
| `variadic_functions.h` | Prototypes des fonctions du projet |
