# 0x0F. C - Recursion

## Description
Résolution de problèmes classiques en C à l'aide de fonctions récursives : affichage, longueur de chaîne, factorielle, puissance, racine carrée, nombres premiers.

## Fichiers

| Fichier | Description |
|---|---|
| `0-puts_recursion.c` | Affiche une chaîne récursivement |
| `1-print_rev_recursion.c` | Affiche une chaîne à l'envers récursivement |
| `2-strlen_recursion.c` | Calcule la longueur d'une chaîne récursivement |
| `3-factorial.c` | Calcule la factorielle récursivement |
| `4-pow_recursion.c` | Calcule une puissance récursivement |
| `5-sqrt_recursion.c` | Calcule la racine carrée récursivement |
| `6-is_prime_number.c` | Vérifie si un nombre est premier récursivement |
