# 0x0C. C - Structures, typedef

## Description
Utilisation des structures et de `typedef` en C à travers un exemple de manipulation d'un objet `dog`.

## Fichiers

| Fichier | Description |
|---|---|
| `dog.h` | Déclaration de la structure `dog` |
| `1-init_dog.c` | Initialise une structure `dog` |
| `2-print_dog.c` | Affiche les informations d'un `dog` |
| `4-new_dog.c` | Crée dynamiquement un `struct dog` |
| `5-free_dog.c` | Libère la mémoire d'un `struct dog` |
