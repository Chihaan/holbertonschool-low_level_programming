# 0x01. C - Variables, if, else, while

## Description
Bases des variables, conditions et boucles `while` en C : nombres positifs/négatifs, alphabet, combinaisons de chiffres, conversions en base 16.

## Fichiers

| Fichier | Description |
|---|---|
| `0-positive_or_negative.c` | Affiche si un nombre est positif, négatif ou nul |
| `1-last_digit.c` | Affiche le dernier chiffre d'un nombre aléatoire |
| `2-print_alphabet.c` | Affiche l'alphabet en minuscule |
| `3-print_alphabet_x10.c` | Affiche l'alphabet 10 fois |
| `4-print_hexa.c` | Affiche les chiffres hexadécimaux |
| `5-print_comb2.c` | Affiche les nombres de 0 à 99 |
| `6-print_comb3.c` | Affiche toutes les combinaisons de 2 chiffres différents |
| `100-*.c` à `102-print_comb5.c` | Exercices avancés de combinaisons de chiffres |
