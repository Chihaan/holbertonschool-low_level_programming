# 0x11. C - Singly linked lists

## Description
Manipulation de listes chaînées simples en C : affichage, longueur, ajout de nœuds, libération de la liste.

## Fichiers

| Fichier | Description |
|---|---|
| `lists.h` | Déclaration de la structure `list_t` et prototypes |
| `0-print_list.c` | Affiche tous les éléments d'une liste `list_t` |
| `1-list_len.c` | Calcule le nombre d'éléments d'une liste |
| `2-add_node.c` | Ajoute un nouveau nœud au début de la liste |
| `3-add_node_end.c` | Ajoute un nouveau nœud à la fin de la liste |
| `4-free_list.c` | Libère une liste `list_t` |
